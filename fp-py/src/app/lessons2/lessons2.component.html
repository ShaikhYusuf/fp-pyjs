Lesson 2: Basic Functional Programming Concepts

1. First-class and Higher-order Functions

First-class Functions:

A first-class function is a function that can be treated like any other value.

  • It can be assigned to a variable.
  • It can be passed as an argument to another function.
  • It can be returned from a function.

Higher-order Functions (HOFs):

A higher-order function is a function that:

  • Takes one or more functions as parameters.
  • Returns another function as a result.
    
      const greet = function(name) {
        return `Hello, ${name}`;
      };
      console.log(greet("Alice")); 
      
      function applyFunction(fn, value) {
        return fn(value);
      }
      console.log(applyFunction(greet, "Bob"));

2. Functions as Arguments and Return Values

JavaScript Example: Function as Argument

    
      function add(x, y) {
        return x + y;
      }

      function operate(fn, a, b) {
        return fn(a, b);  
      }

      console.log(operate(add, 5, 3));

3. Immutability and Data Structures

JavaScript Example: Object.freeze()

    
      const person = { name: "Alice", age: 25 };
      Object.freeze(person);
      person.age = 26;  
      console.log(person.age);

4. Pure Functions

Pure Function Example:

    
      function add(a, b) {
        return a + b;
      }
      console.log(add(2, 3));

5. Lambda Expressions and Anonymous Functions

JavaScript Arrow Functions:

    
      const add = (a, b) => a + b;
      console.log(add(2, 3));

6. Map, Filter, and Reduce

JavaScript Example:

    
      const numbers = [1, 2, 3, 4, 5];

      const squares = numbers.map(x => x * x);
      console.log(squares);

      const evens = numbers.filter(x => x % 2 === 0);
      console.log(evens);

      const sum = numbers.reduce((acc, curr) => acc + curr, 0);
      console.log(sum);
